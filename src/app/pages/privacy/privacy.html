Integritets­policy

Neo Tokyo Kitchen värnar om din personliga integritet. Denna policy beskriver hur personuppgifter kan hanteras i samband med användning av webbplatsens funktioner.

Vilka uppgifter som kan behandlas

När du använder webbplatsen kan personuppgifter behandlas, till exempel vid bordsbokning eller kontakt med restaurangen. Det kan handla om namn, e-postadress, telefonnummer, datum, tid och antal gäster.

Hur uppgifterna används

Uppgifterna används endast för att kunna hantera bokningar, administrera restaurangens tjänster och ge återkoppling vid behov. Uppgifterna används inte för marknadsföring och säljs inte vidare till tredje part.

Lagring av uppgifter

Uppgifter sparas endast så länge det är nödvändigt för det syfte de samlades in för. I detta skolprojekt används databasen främst för att visa hur bokningar och administrativ hantering kan fungera tekniskt.

Cookies och teknisk information

Webbplatsen kan använda teknisk lagring som behövs för att funktioner ska fungera, till exempel inloggning för administratörer. Webbplatsen använder inte cookies för annonsering eller extern spårning.

Dina rättigheter

Du har rätt att få information om vilka personuppgifter som behandlas och att begära rättelse eller radering av uppgifter. Eftersom detta är ett skolprojekt finns inga verkliga kundkonton kopplade till webbplatsen.

Kontakt

Vid frågor om hur personuppgifter hanteras kan du kontakta Neo Tokyo Kitchen via kontaktuppgifterna på kontaktsidan.
